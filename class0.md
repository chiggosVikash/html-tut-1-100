# HTML Elements for Teaching

This document outlines 25 essential HTML elements for teaching an introductory class in HTML.

## Elements

1. **`<html>`**
   - The root element that defines an HTML document.

2. **`<head>`**
   - Contains meta-information about the document.

3. **`<title>`**
   - Sets the title of the document, displayed in the browser's title bar or tab.

4. **`<body>`**
   - Contains the content of the HTML document.

5. **`<h1>` to `<h6>`**
   - Heading elements, with `<h1>` being the highest level and `<h6>` the lowest.

6. **`<p>`**
   - Defines a paragraph.

7. **`<a>`**
   - Defines a hyperlink.

8. **`<img>`**
   - Embeds an image in the document.

9. **`<ul>`**
   - Defines an unordered list.

10. **`<ol>`**
    - Defines an ordered list.

11. **`<li>`**
    - Defines a list item.

12. **`<div>`**
    - Defines a division or section, often used for layout.

13. **`<span>`**
    - Defines a section in a document, used for styling inline elements.

14. **`<form>`**
    - Defines a form for user input.

15. **`<input>`**
    - Defines an input field within a form.

16. **`<meta>`**
    - Provides metadata about the HTML document, such as character set, keywords, and author.

17. **`<link>`**
    - Defines the relationship between the current document and an external resource, often used to link to stylesheets.

18. **`<strong>`**
    - Defines important text, typically rendered in bold.

19. **`<em>`**
    - Defines emphasized text, typically rendered in italics.

20. **`<table>`**
    - Defines a table.

21. **`<tr>`**
    - Defines a row in a table.

22. **`<td>`**
    - Defines a cell in a table.

23. **`<th>`**
    - Defines a header cell in a table.

24. **`<br>`**
    - Inserts a line break.

25. **`<hr>`**
    - Inserts a thematic break (horizontal rule).

These elements cover a wide range of basic functionalities in HTML and will provide a solid foundation for your students.
